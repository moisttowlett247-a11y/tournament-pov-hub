# ⚔️ DMM Allstars Multi-POV Hub

A high-performance, dynamic multi-stream spectator dashboard tailored for tournaments. This hub allows users to orchestrate their own custom viewing experience by pulling multiple live Twitch or Kick streams into a single, cohesive workspace.(Please note that this is a work in progress and is being actively developed to ensure everything works as intended. If you have any features you think would be a good fit for the website please do not hesitate to reach out to me and I will see what can be done to get the features added.)

🌐 **Live Demo:**(https://moisttowlett247-a11y.github.io/tournament-pov-hub/)

---

## ✨ Features

- **Multi-Platform Integration:** Stream seamlessly from both **Twitch** and **Kick** URLs or handles.
- **Dynamic Layout Configurations:** Instantly switch between an optimized **Grid Layout** (multi-view) and a focused **Player Focus Layout** via a single toggle button.
- **Customized Background Profiles:** Personalize your station ecosystem by uploading local media files (JPG, PNG, GIF, WebP, MP4, WebM) or binding direct URLs with integrated canvas loops.
- **Live Chat Integrations:** Select any live competitor's stream panel to hook their chat feed directly into the utility sidebar.
- **Global Deployment Registry:** Register tournament teams and save custom roster configs globally for fast deployments.
